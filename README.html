1. Introduction

This repository contains the procedure of free volume and viscosity calculation by Jeremey Wong.
The main function is test3_pe2.py, and all it does is reading input files and plotting them in two figures.

Also see file /trial/1shell.sh for a more general info about the output of fourier_poly.py, 3body.py and gvdw_2body.py.

2. Input files

Here, we list the content of the files in this repository.

2.1. Files in format dat_{number}K.txt

These files have not being used in the test3_pe2.py or any other function.

2.2. Files in format data_marina{number}.txt

The data in these files are included eta (η\etaη) vs. logMwlogM_wlogMw​ in Figure 1. These files are data from Najm and Savvas, for T=443,483KT = 443, 483KT=443,483K in data_marina.txt and data_marina1.txt.

2.4. Files in format data_{number}K.txt

These files are used in eta (η\etaη) vs. 1000T\frac{1000}{T}T1000​ in Figure 1 where TTT is the temperature in celcius. The columns are TTT and η\etaη. The {number}K in the filename is the molecular weight MwM_wMw​.
These files are from by Najm and Savvas data. η\etaη is in Pa.sPa.sPa.s.

2.5. Files in format pearson_{number}.txt

(except pearson_1994.txt where the 1994 is the year of publication)

These files are used in eta (η\etaη) vs. 1000T\frac{1000}{T}T1000​ in Figure 1. The columns are TTT and log10ηlog_{10}\etalog10​η. The {number} in the filename is the molecular weight MwM_wMw​. The data represents Figure 2 in the Pearson 1987 paper.

2.6. Other files

  • padding_pe is data published by Padding and Briels. It is in format eta (η\etaη) vs. logMwlogM_wlogMw​.
  • pe_dat_ham.txt is data published by Harmandaris et al. . It is in format eta (η\etaη) vs. logMwlogM_wlogMw​.
  • pearson_pe is data published by Pearson 1987. It is in format eta (η\etaη) vs. logMwlogM_wlogMw​.
  • pearson_1994.txtis data published by Pearson 1994. It is in format eta (η\etaη) vs. logMwlogM_wlogMw​.

3. Variables

Except some self explanatory variables such as rho, r, T, N etc. There are variables eta_T, eta_T2,.., eta_T8 that they contain the values for η\etaη based on theory for various molecular weights MwM_wMw​.
These variables contain the input files eta_l_{number}K.txt

This function calculates g(r)g(r)g(r) from PRISM theory. The outputs are in format g-{num1}-{num2}-py3.txt and g-{num1}-{num2}.png (num1 is number of beads and num2 is temperature TTT)
